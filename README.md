# Sugar Caddie V2

Clickable V2 prototype for testing the 18-hole Sugar Caddie round capture, Ready to Play, review, plan and practice loop.

## Vercel settings

Use these settings:

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install

## Updating GitHub

Replace the old repo files with:

- package.json
- vercel.json
- index.html
- src/App.jsx
- src/main.jsx

Make sure the files sit at the top level of the repo, not inside another folder.
